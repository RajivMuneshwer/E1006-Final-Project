Assignment's page.


Assignment 1: Part A was a piece set on psuedocode, binary and truth tables.
Part B was the most challenging assignment which was the generation of lists.


Assignment 2: Part A was the piece set time complexity, iteration and Big O exercise.
Part B had a gravity simulator and making the utilities for the machine learning.


Assignment 3: Had us implement the recursive flood algorithm, and then the completion of the applications for the machinge learning.


Assignment 4: Dealt with Object Oriented Programming. We had to create students, teachers, assignments and classes.
Then figure out their average scores and pay


MIDTERM: Was concerned with the removing interference from three ppm images by taking the mode of each pixel as well as greyscaling and flipping the image.


Assignment 5: Had us finishing off the machine learning of the cancef data.
It involved the use of pandas dataframes and seaborn plots, and finally scikitlearn's machine learning algorithm
